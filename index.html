About

Bio

I am a Front End Developer from Kingston upon Thames, UK.

I have been developing since 2011 and like to build websites / web apps, modern ones that ooze style and functionality and work seamlessly across modern devices. Something that stands out from the crowd as well as being a crowd pleaser.

I'm very much at home working on large traditional sites within cross-functional teams, collaborating our way to success.. Or taking on smaller projects with sole responsibility.

When I'm not helping build the next best thing in code, I spend a portion of my time in Europe coaching snowboarding. This usually being in the winter.

Skills

Skilled in CSS3 (SASS), HTML5 and Javascript (jQuery, AngularJS), I also have good knowledge of CSS Frameworks (Bootstrap), responsive / mobile-first development and CMS integration (Wordpress)

When building a project, no matter the size, I like to keep it organised and moduler. I don't like wasting time either, so tools like Nodel.js, Grunt and Gulp come in handy, especailly when writing in the latest JavaScript (ES6) and being mindful of backward combatability.

For that little extra zing in projects I like to use CSS animations or AngularJS and I'm continuously working on new personal projects to improve both these and other skills (Node.js).
